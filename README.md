<!-- GETTING STARTED -->
## 🚀 Inicio rápido
1.  **Inicia la BD con Docker.**
    Inicia el contenedor de mongo
    ```shell
    docker up
    ```
1.  **Inicializa BE && FE.**
    Navega e inicia.
    ```shell
    #nestjs Backend
    cd backend/
    npm run start::dev
    #reactjs Backend
    cd ../frontend
    npm run start
    ```
1.  **¡Abre el código fuente y empieza a editar!**
    ¡Tu sitio está ahora ejecutándose en `http://localhost:3001`!
